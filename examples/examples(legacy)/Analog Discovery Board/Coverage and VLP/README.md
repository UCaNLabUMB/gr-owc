This folder contains GNU Radio flowgraphs developed for Optical Wireless Communication coverage analysis and Visible Light Positioning (VLP) experiments.

## Flowgraphs

### Power_Detection

Used to detect signal presence and identify periods where a receiver is unable to receive a valid signal.

Applications include:

- Coverage testing
- Signal outage detection
- Receiver availability analysis
- Communication link validation

### Prediction_VLP_Live

Beacon-based Visible Light Positioning (VLP) flowgraph used for localization experiments.

Applications include:

- Indoor positioning
- Beacon detection
- Location estimation
- Real-time VLP testing
