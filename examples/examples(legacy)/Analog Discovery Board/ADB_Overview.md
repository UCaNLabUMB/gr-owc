# Analog Discovery Board

This folder contains GNU Radio flowgraphs, Python scripts, and documentation developed for Optical Wireless Communication (OWC), Visible Light Positioning (VLP), and signal processing experiments using the Digilent Analog Discovery platform.

## Contents

* **Communications** – OFDM-based OWC and hybrid RF/OWC communication flowgraphs.
* **Basic Tx Rx** – Signal generation, loopback, and hardware verification examples.
* **Power Detection Prediction** – Signal detection, power measurement, SINR analysis, and data collection tools.

## Requirements

* Analog Discovery 2 or 3
* GNU Radio 3.10
* WaveForms Software
* WaveForms SDK

Some flowgraphs may require updating file paths and SDK locations before use.
